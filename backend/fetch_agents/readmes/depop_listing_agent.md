# Depop Listing Agent

## Description

Specialized Depop listing assistant that turns analyzed resale inventory into a draft listing ready for review or submission.

## Use Cases

- Generate a Depop draft from an item description and photos.
- Revise listing tone and copy before publish.
- Prepare a listing workflow for live Browser Use submission.

## Example Prompts

- Turn this item into a Depop draft.
- Create a Depop listing for this Carhartt jacket.
- Revise the draft to sound more streetwear-focused.

## Input Requirements

- A sell-side item description.
- Optional image URLs.
- Optional listing instructions or style notes.

## Output Summary

- Listing title and description.
- Suggested price and category path.
- Draft status and any confirmation-ready state.

## Limitations

- Live publish behavior depends on Browser Use environment readiness.
- Draft quality depends on upstream identification and pricing quality.
- Built for Depop, not general multi-marketplace listing.

## Differentiation

Use this agent when the goal is to create or revise a Depop listing rather than identify or price an item.
