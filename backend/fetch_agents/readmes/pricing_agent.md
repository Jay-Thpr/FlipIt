# Pricing Agent

## Description

Specialized resale pricing agent that combines item identification and sold-comps analysis into a list-price and profit recommendation.

## Use Cases

- Estimate a realistic list price.
- Calculate expected profit before fees and sourcing decisions.
- Produce a pricing recommendation before drafting a listing.

## Example Prompts

- What should I price this vintage Nike tee at?
- Estimate profit on this jacket if I list it now.
- Price this item from the photos and notes.

## Input Requirements

- A natural-language item description.
- Optional image URLs.
- Optional extra condition or sourcing notes.

## Output Summary

- Recommended list price.
- Expected profit.
- Pricing confidence and short rationale.

## Limitations

- Pricing quality depends on the sold-comps and vision heuristics.
- Does not submit listings or negotiate with buyers.
- Market timing and trends are simplified.

## Differentiation

Use this agent when the user already knows the item and wants a resale valuation.
