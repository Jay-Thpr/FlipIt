## PART 5: Side Track Stacking Strategy

### How to Stack Maximum Prizes

The side tracks (Solo, Duo, Beginner, AI/ML, UI/UX, Mobile) stack on top of everything. Here's how to think about them:

**Best AI/ML Hack** — Virtually any project with a sponsor API qualifies. TwelveLabs, Gemini, ElevenLabs are all ML. Make sure your README highlights the ML components.

**Best UI/UX Hack** — Invest 2-3 hours in polish. Use a clean design system (shadcn, Tailwind). Smooth animations. Good typography. This is often the difference between winning and not.

**Best Mobile Hack** — If your core idea works, build a React Native or Flutter wrapper. Even a basic mobile interface gets you eligible. Pairs well with Health and Commerce tracks.

**Best Duo Hack** — If you end up as a two-person team, this is free eligibility. 27" monitor is a solid prize.

**Best Solo Hack** — Apple Watch SE 3 is a great prize. Only matters if you go solo.

### The Ideal Prize Stack
A dream scenario: **1st Place Overall + Browser Use (iPhones + SF trip) + a Main Track Prize + Best AI/ML + Best UI/UX**. That's 5 separate prizes from one project.

To achieve this, you need a project that:
1. Uses Browser Use as core functionality
2. Fits cleanly into one main track
3. Has a clear ML/AI component
4. Looks beautiful and feels intuitive
5. Is impressive enough for an overall prize

### Meta-Strategy Notes

- **Browser Use has the highest-value prizes by far** (iPhones + AirPods + SF trip). Weight your project toward this if prize value is a priority.
- **Interactive AI (iPad Air)** is the single most valuable individual sponsor prize. If you don't go Browser Use, this is the next target.
- **Fetch.ai cash prizes are modest** ($300/$200) but easy to stack since it just requires Agentverse registration.
- **TwelveLabs prizes are credits, not hardware** — good for learning, lower prize value. But TwelveLabs makes demos impressive (video is visual and engaging for judges).
- **.Tech domain is essentially free** — register your project domain regardless. Desktop mic + 10 years of a domain.
- **Vultr is free infrastructure** — claim the cloud credits and use them whether or not you're targeting the prize.
- **ElevenLabs and Gemini prizes are modest** (earbuds, swag kits) but adding voice or Gemini to any project is usually low effort.
- **Judges remember demos that surprise them.** A niche idea with a clean demo beats a broad idea with a clunky one every time.
